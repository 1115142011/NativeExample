#### 明确的类型可以暗示正在处理特殊数据以减少问题发生的概率 Int 表示的是有符号的整数
- 在 swift 中 数据类型的转换必须显式的指定 
```swift
let three = 3
let pointOneFourOneFiveNine = 0.14159
let pi = Double(three) + pointOneFourOneFiveNine
// pi 等于 3.14159，所以被推测为 Double 类型

```
